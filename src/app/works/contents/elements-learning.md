プログラミング言語実験の JavaScript 回で作成した元素記号を覚えるためのサイトです。

「記号 → 日本語名」「記号 → 英語名」「日本語名 → 記号」「英語名 → 記号」
の 4 種類のクイズに答えることができ、その結果は Cookie に記録されます。
この記録を用いて周期表から正誤を確認することができます。
